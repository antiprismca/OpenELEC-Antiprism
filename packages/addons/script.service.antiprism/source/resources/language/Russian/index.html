    Конфигурирование Вашего браузера для использования с АнтиПризмой (через Privoxy):

  • Откройте установки прокси
  • Введите IP адрес Вашей АнтиПризмы в полях адресов прокси-сервера для HTTP и HTTPS
  • Введите порт 8118 для HTTP и HTTPS
  • Удостоверьтесь, что "Контроль доступа" АнтиПризмы разрешает доступ к прокси с IP адреса Вашего браузера
  • Наслаждайтесь анонимностью!
