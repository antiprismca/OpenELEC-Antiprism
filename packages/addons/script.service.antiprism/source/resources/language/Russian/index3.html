    Конфигурирование Вашего браузера для использования с АнтиПризмой через сеть I2P:

  • Откройте установки прокси
  • Введите IP адрес Вашей АнтиПризмы в поле адреса прокси-сервера
  • Выберите порт 4444/HTTP, порт 4445/HTTPS
  • Удостоверьтесь, что "Контроль доступа" АнтиПризмы разрешает доступ к прокси с IP адреса Вашего браузера
